# MONGO_GRID_FS__2__AWS_S3
Stream file from MongoGridFS to AWS S3 Bucket

## HOW WORK
- Create file __.env__ under _environment_ directory, according to _.env.example_ and set variables.
- Run npm run dev
- Visit ___http://localhost:___<PORT_ENV_VARIABLE>___/api-doc for swagger docs.
- App has a frontend too. Visit ___http://localhost:___<PORT_ENV_VARIABLE>___/
